Consider the following statement:

"Dyson air purifiers are the best."

Which set of hypotheses could accurately test this statement?

SOLUTION: Null: Dyson air purifiers **are not** the best. Alternative: Dyson air purifiers **are** the best.

Consider the following statement:

"The average height of a human is smaller than or equal to 70 inches."

Which set of hypotheses below is correctly set up based on this assertion?

SOLUTION: Null: The average height of a human is not greater than 70 inches. Alternative: The average height of a human is greater than 70 inches.
